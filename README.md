# Dosia

ПО для нагрузочного тестирования HTTP/HTTPS ресурсов. 

## Установка на MacOS/Windows
```shell
1. Загрузите приложение по одной из ссылок ниже
 
2. Запустите распакованный файл

3. Следите, чтобы компьютер не уходил в "сон"
```
